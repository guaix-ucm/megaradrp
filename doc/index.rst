MEGARA DRP documentation
========================

Welcome. This is the Documentation for MEGARA DRP (release |release|).

MEGARA (Multi-Espectrógrafo en GTC de Alta Resolución para Astronomía) is 
an optical Integral-Field Unit (IFU) and Multi-Object Spectrograph (MOS) 
designed for the GTC 10.4m telescope in La Palma.

MEGARA DRP is the official Data Reduction Pipeline for MEGARA. It can
be run as a standalone module (via a command line interface) or
integrated in the GTC Control System. 

Although a MEGARA Cookbook is accesible in PDF format through `GTC <https://www.gtc.iac.es/instruments/megara/media/MEGARA_cookbook_1I.pdf>`_ and 
`Zenodo <https://zenodo.org/records/3932063>`_, the most up-to-date 
version of it, including a description of 
additional MEGARA tools, can be found online at 
this `link <https://guaix-ucm.github.io/megaradrp-cookbook/>`_.
